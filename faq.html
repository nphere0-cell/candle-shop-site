NPHere - FAQ

Q1: Mujhe order kaise karna hai?

Aap website pe product select karo, add to cart karo aur payment complete karo. Bas, order confirm ho jayega.

Q2: Payment ke kaun kaun se options hain?

Credit card, debit card, UPI, net banking, aur cash on delivery sab available hai.

Q3: Delivery kitne din mein hoti hai?

Location ke hisab se 3 se 7 din ke andar delivery ho jaati hai.

Q4: Return policy kya hai?

Agar product mein defect ho ya galat item aaye, toh aap 7 din ke andar return request kar sakte ho.

Q5: Contact kaise karun agar help chahiye?

Aap hamare contact form, email ya customer support number se directly help le sakte ho.
